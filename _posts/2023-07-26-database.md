### MSSQL 

#### 테이블 생성해서 데이터 복사
```sql
SELECT * INTO 생성테이블 FROM 원본테이블
```

#### 데이터만 복사
```sql
INSERT INTO 데이터 입력 테이블명 SELECT * FROM 원본 테이블
```
