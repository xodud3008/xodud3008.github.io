#### AWS Ubuntu npm install 중 에러 발생
```
Error: Missing required argument #1
```

npm 버전이 낮아서 생기는 오류이다.

```
# npm cache clean --force
# npm install -g n
# n stable
```

npm을 최신 버전으로 설치하면 해결된다.
최신 버전 설치 후에도 안된다면 서버를 껐다 다시 켜보자.
